# NATURIA SMART-SKIN EDU
NATURIA adalah inovasi face wash ramah lingkungan yang memadukan ekstrak daun teh hijau (Camellia sinensis L.) sebagai sumber antioksidan dan arang aktif dari tempurung kelapa (Cocos nucifera L.) untuk adsorpsi kotoran dan toksin kulit.
Produk ini dipadukan dengan platform edukasi dan rekomendasi perawatan kulit berbasis AI SmartSkin-Edu
