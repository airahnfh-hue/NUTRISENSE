# NUTRISENSE
NUTRISENSE merupakan inovasi piring pintar yang menggabungkan teknologi Artificial Intelligence dan sensor IoT untuk membantu pengguna memahami pola makan secara real-time.  Dengan sistem pengenalan makanan otomatis, pengukuran berat, dan rekomendasi nutrisi personal, NUTRISENSE menjadikan kebiasaan makan lebih sehat, efisien, dan cerdas.
